# OpenCode + MongoDB Starter

AI-powered coding workspace with CodeLlama 34B via Ollama.

## Quick Start

```bash
npm install
npm start
```

## Using OpenCode

```bash
opencode
```

Then just describe what you want to build!

## Configuration

- `opencode.json` - Pre-configured for Ollama with CodeLlama 34B
- MongoDB available at `localhost:27017`
